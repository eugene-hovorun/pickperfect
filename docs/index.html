PickPerfect — Privacy Policy

Last updated: February 7, 2026

The short version

PickPerfect does not collect, transmit, or share any data. Period. Everything stays on your device.

What data does PickPerfect store?

PickPerfect stores the following data locally on your device using Chrome's built-in storage API (chrome.storage.local):

  • Color history — the last 20 colors you picked (HEX values and timestamps)
  • Format preference — your preferred color format (HEX, RGB, or HSL)

This data never leaves your browser. It is not transmitted to any server, third party, or external service.

What data does PickPerfect collect?

None. PickPerfect does not collect any personal data, usage data, analytics, or telemetry of any kind.

Permissions

PickPerfect requests a single Chrome permission:

  • storage — to save your color history and format preference locally on your device

PickPerfect does not request access to your browsing history, tabs, web requests, or any website content. It uses the native EyeDropper API, which requires no content script injection.

Third parties

PickPerfect does not use any third-party services, SDKs, analytics tools, or advertising networks.

Changes to this policy

If this policy changes, the updated version will be published at this same URL with an updated date.
