PickPerfect — Privacy Policy

Last updated: February 7, 2026

The short version

PickPerfect does not collect, transmit, or share any personal data. Everything stays on your device. Premium features use ExtensionPay for payment processing only.

What data does PickPerfect store?

PickPerfect stores the following data locally on your device using Chrome's built-in storage API (chrome.storage.local):

  • Color history — the last 20 colors you picked (HEX values and timestamps)
  • Format preference — your preferred color format (HEX, RGB, or HSL)
  • Premium status — whether you've purchased premium features (cached locally for faster popup loading)

This data never leaves your browser. It is not transmitted to any server, third party, or external service (except ExtensionPay for payment verification — see below).

What data does PickPerfect collect?

None. PickPerfect does not collect any personal data, usage data, analytics, or telemetry of any kind.

Permissions

PickPerfect requests the following Chrome permissions:

  • storage — to save your color history and format preference locally on your device
  • activeTab — to extract colors from the current webpage when you click "Extract Palette" (premium feature only, on-demand)
  • scripting — to inject the palette extraction script when you click "Extract Palette" (premium feature only, runs once then disappears)

PickPerfect does not request access to your browsing history or any content beyond the active tab when you explicitly click the extraction button. It uses the native EyeDropper API for core color picking, which requires no content script injection.

Premium features (optional)

If you purchase premium features ($2.99 one-time), PickPerfect gains access to:

  • WCAG Contrast Checker — runs locally, no data transmitted
  • Tailwind Color Mapping — runs locally, no data transmitted
  • Page Palette Extraction — reads computed CSS colors from visible elements on the active tab when you click "Extract Palette". This data is processed locally and never sent to any server.

The palette extraction feature only runs when you explicitly click the button. It extracts color values from CSS styles and does not access page content, form data, cookies, or any personal information.

Third parties

ExtensionPay (for premium features only):

  • Used for payment processing and premium feature verification
  • Processes payment information through Stripe
  • Their privacy policy: https://extensionpay.com/privacy

PickPerfect does not use any analytics tools, advertising networks, or other third-party services.

What PickPerfect does NOT do

  • ❌ No tracking or analytics
  • ❌ No data collection
  • ❌ No ads or advertising networks
  • ❌ No cookies
  • ❌ No form data access
  • ❌ No browsing history
  • ❌ No background activity (except ExtensionPay verification)
  • ❌ No persistent content scripts
